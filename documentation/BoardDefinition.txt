The board is a container for tables defined by a user. 
Each table can be empty or it can contain tasks.
The board has a backgroud of user's choice. He can import
it from the internet only.

Each board belongs to a user. In addition, a board has
its own configuration. A configuration contains following settings:
- name
- background image,
- owner,
- creation date,
- start date,
- finish date,
- status (active, finished)

A user has a list of boards that are grouped by the status:
- active
- finished
